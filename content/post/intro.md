## 🛠 技术能力储备

### 1. 后端架构与高并发处理
- **技术栈：** 深入研究并广泛应用 Linux, Shell, Go, Python, PHP, C/C++。
- **擅长领域：** 解决多进程、多线程环境下的复杂并发问题。擅长设计任务调度系统、优雅退出机制以及异常回滚策略，确保生产环境的系统鲁棒性。
- **近期实战：** 针对大规模数据校验任务，重构了基于分布式信号同步的并行处理系统，实现了 API 额度自动感知与任务状态的原子性切换。

### 2. 数据采集与 ETL 工程
- **技术栈：** 熟练运用多种采集框架，精通 PostgreSQL 优化与复杂数据结构化处理。
- **擅长领域：** 构建高性能、高自动化的数据采集与清洗链路。能够处理海量请求调度、反爬避让以及大规模数据的存储与索引优化。

### 3. 全栈开发与自动化方案
- **技术栈：** 前端主流框架（React/Vue）、后端微服务架构。
- **擅长领域：** 开发响应式 Web 应用、企业级办公自动化工具（Python Office 自动化）、以及定制化的内部管理系统。

### 4. 网络工程与链路优化方案
- **技术实践：** 熟练运用 frp, gost, tailscale 等工具实现复杂环境下的内网穿透与异地组网。
- **擅长领域：**
  - **高级内网穿透**：保障各种私网环境下服务的稳定映射与安全访问，解决异地办公的连通性痛点。
  - **全球链路加速与优化**：针对国际化业务场景下的链路延迟与访问受限问题，提供高效的全球化互联优化方案，提升跨境协作与数据同步效率。
